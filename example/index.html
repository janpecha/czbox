Basic

Load Zepto.JS and CzBox

Load Zepto.JS:

<script type="text/javascript" src="zepto.js"></script>

Load CzBox2:

<script type="text/javascript" src="./czbox/czbox2.js"></script>

Set CzBox skin

<link rel="stylesheet" type="text/css" href="./czbox/czbox.css">

Add image to HTML document

Images must have set attribute rel="czbox"

<a href="photos/foto1_thumb.jpg" rel="czbox">
	<img src="photos/foto1_big.jpg">
</a>

or alternatively rel="lightbox"

<a href="photos/foto1_thumb.jpg" rel="lightbox">
	<img src="photos/foto1_big.jpg">
</a>

Run CzBox

<script type="text/javascript">
	Zepto(function($){
		CzBox.create();
	});
</script>

or only (for script on end of document):

<script type="text/javascript">
	CzBox.create();
</script>

Create roadmaps

We use rel="czbox[myRoadMapID]" instead of rel="czbox":

<a href="photos/foto1_thumb.jpg" rel="czbox[myroadmap]"><img src="photos/foto1_big.jpg"></a>
<a href="photos/foto2_thumb.jpg" rel="czbox[myroadmap]"><img src="photos/foto2_big.jpg"></a>
<a href="photos/foto3_thumb.jpg" rel="czbox[myroadmap]"><img src="photos/foto3_big.jpg"></a>

Image description

<a href="photos/foto1_thumb.jpg" rel="czbox">
	<img src="photos/foto1_big.jpg" title="this is description">
</a>

or

<a href="photos/foto1_thumb.jpg" rel="czbox" title="this is description">
	<img src="photos/foto1_big.jpg">
</a>

or

<a href="photos/foto1_thumb.jpg" rel="czbox">
	<img src="photos/foto1_big.jpg" alt="this is description">
</a>
